## How to use 
```cmd
python3 check_and_reply.py --judge_rst <the judge result> --question <question path> --code <want to check code> --session <config.ini filepath> --lang <checked code using language> --save <save ChatGpt return result>
```
or
```python
import check_and_reply

check_and_reply.run(
    judge_rst="",
    code="",
    session="",
    code="",
    lang="",
    save=""
)
```
